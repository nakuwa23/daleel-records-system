
# Daleel — Portable Digital Academic Records System

An offline-aware system that lets authorized schools issue verified, tamper-resistant academic records that follow displaced and migrant learners and can be verified offline by any receiving school.

## Structure

- `backend/` — Django + DRF API (records, identity, verification, cryptographic signing)

- `frontend/` — Next.js PWA

## Stack

Django REST Framework · PostgreSQL · Ed25519 signatures · JWT auth · Next.js PWA
